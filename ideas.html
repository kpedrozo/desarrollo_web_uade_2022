Las mejores ideas para construir un sitio web en armonia

Armonia del color

En todas las ramas del diseño el color es uno de los elementos claves para que el resultado final resulte coherente y agredable a la vista. Podemos encontrar facilmente paletas de colores para jugar con el contraste y la armonia.La armonía es esencial ya que si han de relacionarse entre si todos los colores de una composición, deben ajustarse a un todo unificado. Existen objeciones mas fundamentales al principio sobre el cual se basan las reglas de la armonía del color. Este principio concibe una composición de colores como un conjunto donde todo se ajusta a todo. Los diferentes tipos de composiciones nos van a permitir lograr transmitir el sentimiento de nuestro sitio web.

Tipografía

La correcta elección de la tipografía inside directamente sobre la percepción en la que el usuario capta la informacion. Una buena tipografía hace que el texto sea legible. Si el texto es difícil de leer, los usuarios abandonarán la página. Al igual que cualquier otro elemento del diseño, la tipografía implica una forma de comunicar. Tal es así, que incluso se puede imaginar que cada fuente tiene su propia voz. Por eso, debemos elegir el tipo de letra que mejor represente los valores de nuestra marca. Una buena tipografía puede definir, por sí sola, la estética de casi todo un sitio web.

Etiquetas HTML

Nuestro sitio web se compone además del texto e imágenes, de etiquetas, llamadas TAGS, lo que nos permite darle la estructura necesaria para hacer del maquetado no solo una idea, sino convertirla en algo funcional. Hay diferentes tipos de etiquetas y un correcto uso de las mismas nos ayuda a posicionar el sitio en los motores de optimización de buscadores. Con las etiquetas podemos darle niveles de lectura, también separar en secciones, identificar tanto la barra de navegación como el pie de la página, incluir contenido que se verá en desktop y se ocultará en versión mobile.

Maquetado

Para lograr una vista previa de nuestro sitio web, sin todo el esfuerzo que conlleva el desarrollo del código, podemos hacer uso de una de las técnicas más efectivas, que nos van a ahorrar mucho tiempo, diseñar un prototipo, posicionando cada uno de los elementos en sus respectivos lugares. Para esto podemos usar desde papel y lápiz, hasta herramientas de uso colaborativo y gratuitas para que, una vez aceptado el modelo, avancemos con la programación.
